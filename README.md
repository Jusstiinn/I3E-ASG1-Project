# Overview

### Description
This is a short fast-paced game about escaping a cursed factory, where the exit gate is locked tight by magic crystals scattered around the establishment. The player has to find and break all the crystals for the gate to be realeased so that they can escape. However, things aren't that straightforward, theres a evil presence roaming the establishment trying to stop the player from escaping.

### Minimum System Requirements
OS: Windows 10/11 64-bit
CPU: Intel Core i5-10400 / AMD Ryzen 5 3600
RAM: 8 GB
GPU: NVIDIA GTX 1650 / AMD RX 580
Storage: 5 GB SSD space
Display: 1920px x 1080px, 60hz

### Controls
Use W,A,S,D to move forwards, left, backwards and right respectively.
press Spacebar to jump.
Use E to interact with interactable objects
Use a mouse to pan the camera and look around.

### How to beat the game
Take advantage of the extra jumping mobility you have, ramps and sharp turns to avoid the entity. The entity cannot jump off elevated surfaces.
To unlock the locked door, you have to do a short parkour section where falling off means death. The key sits at the end of the parkour section.  
To open the exit gate, you have to collect all 55 crystals.
